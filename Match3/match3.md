# Match-3 Game

This is a simple Match-3 game implemented using HTML, CSS, and JavaScript. In this game, your goal is to match three or more tiles of the same color in a row or column to earn points. The game also includes a timer, and your objective is to achieve the highest score before the time runs out.

## Technologies Used

- HTML: Used for the structure of the game.
- CSS: Used for styling and animations.
- JavaScript: Used for game logic and interactivity.
- Canvas: Used for the game board.

## How to Play

1. Click the "Start Game" button to begin.
2. You'll see a grid of colored tiles.
3. Click on a tile to select it. Click on another adjacent tile to swap their positions.
4. If the swap results in a match of three or more tiles of the same color in a row or column, those tiles will disappear, and you'll earn points.
5. The game includes a timer, and you must try to earn as many points as possible before the time runs out.
6. Keep matching tiles and earning points to achieve a high score.
7. If there are no more possible moves, the game is over.

## Features

- Colorful and visually appealing design.
- Tiles with animations and hover effects to make the gameplay engaging.
- Keeps track of your score and high score through local storage.
- A timer adds a time-based challenge to the game.
- Game over screen with the option to restart the game.
- Background music (audio).

Have fun playing the Match-3 game!
